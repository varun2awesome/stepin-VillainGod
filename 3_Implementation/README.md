# Implementation

## Folder Structure
Folder        | Description
--------------| ----------------------------------------------
`inc`         |  header files
`src`         | Source code files for VillainGod
`test`        | All source code and data for testing purposes
`unity`       | Files for unit testing


### To build and execute:
```
make all
make run
```





### Documentation

```
make doc
```

### Test

```
make test
```

### Coverage

```
make coverage
```

make dynamic
```

### Uninstall
```
make clean
```
